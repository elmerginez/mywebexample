Problema propuesto 2

La evaluacion de un estudiante se basa en 6 notas, de las cuales se consideran solo las 4 mejores notas 


                    // declaracion de variables
                    int notas[] = {15,12,18,17,14,8};
                    int promedio = 0;
                    
                    // for para ordenar el vector de mayor a menor
                    for (int i = 0; i < notas.length-1; i++) {
                        if (notas[i] < notas[i+1]) {
                            int temp = notas[i];
                            notas[i] = notas[i+1];
                            notas[i+1] = temp;
                        }
                    }
                    
                    //for para sumar los 4 numeros del vetor
                    for (int i = 0; i < 4; i++) {
                        promedio += notas[i];
                    }
                    promedio /=4;// divide la suma
                    
                    // imprime el promedio
                    System.out.println("Promedio: " + promedio);
                    //fin programa

Problema propuesto 4

Desarrollar un programa que permita el ingreso de "N" numeros enteros positivos.
Luego debe encontrar la media y la mediana.


                    // instaciacion del objeto entrada de la clase Scanner
                    Scanner entrada = new Scanner(System.in);
                    // declaracion de variables
                    int numeros[] = new int[5];
                    int mediana = 0;
                    int media = 0;
                    
                    // for para pedir los numeros positivos
                    for (int i = 0; i < numeros.length; i++) {
                        System.out.println("Ingresa numero positivo: ");
                        numeros[i] = entrada.nextInt();
                    }
                    
                    // for para sumar los numeros del vector
                    for (int i = 0; i < numeros.length; i++) {
                        media += numeros[i];
                    }
                    media /=numeros.length;//divide la suma total del vector por el tamano del vector
                    
                    // ordenar el vector de menor a mayor
                    Arrays.sort(numeros);
                    mediana = numeros[numeros.length/2];
                    
                    // imprime la media y la mediana del vector
                    System.out.println(media);
                    System.out.println(mediana);
                    //fin programa

Problema propuesto 5

Desarrollar un programa que genere un arreglo de 15 numeros enteros entre 20 y 30, luego debe encontrar la moda
El programa debe mostrar el arreglo y la moda


                    // declaracion de variables
                    int moda = 0;
                    int totalRep = 0; 
                    int[] arreglo = new int[15];
                    
                    // ellenar el arreglo con n√∫meros aleatorios entre 20 y 30
                    for (int i = 0; i < arreglo.length; i++) {
                        arreglo[i] = (int) (Math.random() * 11) + 20;
                    }
            
                    // for para encontrar la moda
                    for (int i = 0; i < arreglo.length; i++) {
                        int rep = 0;//variable local
                        
                        // for para calcular las repeticiones de un numero
                        for (int j = 0; j < arreglo.length; j++) {
                            if (arreglo[i] == arreglo[j]) {
                                rep++;
                            }
                        }
                        
                        // si repeticion es mayor a total de repeticiones 
                        // el nuevo valor de total de repeticiones es al de repeticion
                        // y moda es igual al valor del numero que se repite
                        if (rep > totalRep) {
                            totalRep = rep;
                            moda = arreglo[i];
                        }
                    }
                    
                    // imprimir el arreglo y la moda
                    System.out.println("arreglo: "+Arrays.toString(arreglo));
                    System.out.println("moda: " + moda);
                    // fin programa

Problema propuesto 6

Desarrollar un programa que genere una matriz de 4x3 de numeros aleatorios comprendidos entre 10 y 30.
Luego debe calcular la suma de cada columna.
Finalmente, el programa debe mostrar la matriz y la suma de sus columnas.


                    // declaracion de variables
                    int matriz[][] = new int[4][3];
                    int suma = 0;
                    
                    // for para rellenar la matriz de numeros aleatorios
                    for (int i = 0; i < matriz.length; i++) {
                        for (int j = 0; j < matriz.length-1; j++) {
                            matriz[i][j] = (int) (Math.random() * 21) + 10;
                            System.out.print(matriz[i][j] + "  ");
                        }
                        System.out.println("");
                    }
                    System.out.println("----------");// para dividir los datos y la suma :D
                    
                    // for para sumar cada columna de la matriz
                    for (int i = 0; i < matriz.length-1; i++) {
                        for (int j = 0; j < matriz.length; j++) {
                            suma += matriz[j][i];
                        }
                        System.out.print(suma + "  ");
                        suma = 0;
                    }
                    System.out.println("");
                    //fin programa
